# chatbot-function-calling
A Python chatbot that uses Claude's function calling feature to get real time weather data for any city in the world.

What is Function Calling?
Normal chatbot → Claude answers from memory only
This chatbot → Claude calls real functions to get live data!

Example:
You: What is the weather in Chennai?
Bot: Calls weather function → Gets real data → Answers! ✅

## 🛠️ Libraries Used
- anthropic
- requests
- python-dotenv
